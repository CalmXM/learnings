---

# 环境说明

1、Ubuntu: Ubuntu 20.04.2 LTS

# 安装步骤 

下面根据 Redis 官网的安装说明 [ From source code](https://redis.io/download#from-source-code) 进行操作。

```
$ cd /tmp
$ wget https://download.redis.io/releases/redis-6.2.1.tar.gz
$ tar xzvf redis-6.2.1.tar.gz
$ cd redis-6.2.1
$ make
```
